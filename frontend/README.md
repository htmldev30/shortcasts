## Frontend | Run Locally

**Backend Does Not Have A Hosted Version For Development**
<br />
**_No User Auth_**

Clone the project

```bash
  git clone
```

Go to the project directory

```bash
  cd frontend
```

Install dependencies

```bash
  yarn
```

Start the server

```bash
  yarn dev
```
