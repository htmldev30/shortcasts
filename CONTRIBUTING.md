# Contributing to ShortCasts

## Backend | Run Locally

<p align="center">
  <strong>Node & ExpressJS</strong>
</p>

Clone the project

```bash
  git clone
```

Go to the project directory

```bash
  cd backend
```

Install dependencies

```bash
  yarn
```

Start the server

```bash
  yarn dev
```

---

**Backend Does Not Have A Hosted Version For Development** <br />
**_No User Auth_**

## Frontend | Run Locally

<p align="center">
